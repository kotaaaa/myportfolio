# Usage

## Install Dependencies
Run following commands in the root directory. node version is `v18.16.0`
```
yarn install 
```

## Start app
Run following commands and open http://localhost:3000 in your browser.
```
npm run dev
```

*  CSS
*  Tailwind CSS
*  Flowbite
