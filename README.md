# my portfolio site

# Usage

## Install Dependencies
Run following commands in the root directory. node version is `v16.20.0`
```
yarn install 
```

## Start app
Run following commands and open http://localhost:3000 in your browser.
```
yarn dev
```

# Tech stack
*  HTML
*  CSS
*  Tailwind CSS
*  Flowbite
*  Next.js
*  Typescript
